Quick Guide

Draw

Choose a tool, click-drag on the sheet, then release. Doors, windows, text and library symbols are click-to-place.

Edit

Use Select, click an item, drag to move it, or edit precise values in Properties.

Navigation

Mouse wheel zooms. Middle mouse or Pan tool moves the view. Delete removes selection.

Storage

Save writes project data only to this browser's localStorage. Export JSON for portable backups.

This is a capable browser-based 2D CAD foundation, not a certified engineering, BIM, parametric-solid, or manufacturing CAD system. Verify dimensions before construction.
